# 🏖️ SandyVault

**SandyVault** is a beautifully animated, fully responsive frontend e-commerce simulation platform for digital goods such as Courses, Ebooks, Bundles, and Projects. Built entirely with **HTML**, **CSS**, and **Vanilla JavaScript**, it showcases DOM manipulation, form validation, dynamic rendering, and persistent data handling via `localStorage`.
